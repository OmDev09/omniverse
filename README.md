# Omniverse - Web Development Agency

Omniverse is a modern, premium, and high-conversion landing page designed for a forward-thinking digital agency. It utilizes a futuristic "dark-tech" aesthetic, prioritizing dynamic user experiences through smooth scroll reveals, interactive glassmorphism components, and custom canvas-based particle animations.

## ✨ Features

- **Cinematic Intro Sequence**: A custom text-shatter animation that builds anticipation before revealing the main site.
- **Interactive Canvas Background**: A constantly evolving network of particles rendered via HTML Canvas, representing the "Omniverse" of digital connectivity.
- **Scroll Reveal Animations**: Smooth, buttery fade-up transitions for all major content blocks as the user scrolls down the page.
- **Glassmorphism Design System**: Dynamic frosted-glass panels (`backdrop-filter`) that beautifully blur the network background to create depth.
- **Bento Box Layouts**: Modern asymmetrical grid layouts used for the "Omniverse Advantage" section.
- **Intersection Observer Counters**: Number statistics that animate natively when they enter the user's viewport.
- **Mobile Responsive**: Fully fluid layout with a dedicated mobile hamburger menu.

## 🛠️ Technology Stack

- **HTML5**: Semantic markup structuring.
- **Vanilla CSS3**: Utilizing CSS Variables for theming, CSS Grid & Flexbox, Keyframes, and native CSS filters. No external CSS frameworks are used, ensuring maximum performance and customizability.
- **Vanilla JavaScript**: Handling DOM manipulation, canvas rendering, Intersection Observers, and scroll logic without heavy dependencies.
- **Vite**: Ultra-fast frontend build tool ensuring instant hot-module-replacement (HMR) during development.

## 🎨 Design Aesthetic

The website leans heavily into a high-end tech/SaaS agency feel:
- **Primary Colors**: Deep Space Black (`#030712`) and Slate Blue (`#0f172a`).
- **Accents**: Electric Blue (`#3b82f6`) and Sky Blue (`#0ea5e9`).
- **Typography**: Clean, readable sans-serif typography (`Inter` font).
- **Textures**: Soft radial glowing gradients and semi-transparent frosted glass panels.

## 🚀 Getting Started

To run this project locally, you will need [Node.js](https://nodejs.org/) installed on your machine.

1. **Install Dependencies**:
   Open your terminal in the project directory and run:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```
   This will start the Vite development server. Open the provided `localhost` URL in your browser.

3. **Build for Production**:
   ```bash
   npm run build
   ```
   This command creates an optimized, minified bundle in the `dist` directory, ready to be deployed.
